# Product

## Register

brand

## Users

AI engineers, agent developers, and LLM power-users who build agent systems and need a memory layer that doesn't cost API calls. They ship code, they know the filesystem, and they're tired of paying per-query for LLM-driven memory janitors. The landing page visitor is evaluating Strata against mem0, LangChain memory, or their own hacky solution.

## Product Purpose

Strata is a CLI-first, stdlib-first tiered memory system for AI agents. It layers memories by age across three strata (active, cooled, archive) using algorithmic triggers instead of LLM calls. The page's job: make the geological metaphor land, communicate the zero-LLM-cost insight, and drive mailing list signups for early access.

## Brand Personality

Precise, geological, anti-hype, CLI-native, practical. Strata sounds like a geologist wrote it, not a marketer. The voice trusts the reader's intelligence. No exclamation points. No startup buzz. Descriptions are compact and physical: files settle, sediment cools, shadows index. The brand has the calm confidence of something that actually works.

## Anti-references

The page must avoid all three:
- Generic SaaS landing pages (gradient hero, card grid, testimonials carousel)
- Dark-mode-by-default dev tool aesthetic (no neon on black)
- Editorial-typographic minimalism (no italic serif displays, ruled columns, magazine affectation)

The Strata landing page should be unlike a typical landing page — it should feel more like a specimen, a cross-section, a geology display.

## Design Principles

1. **The metaphor is the structure** — every design decision traces back to geological strata. Layers, cracks, sediment, cross-sections. If it doesn't serve the metaphor, reconsider.
2. **Proof over promise** — show the CLI output, show the latency table, show `strata init` succeeding. The page earns trust through specificity, not testimonials.
3. **No AI slop** — no gradient text, no glassmorphism, no hero metrics, no identical card grids. The page demonstrates the exact quality it claims for Strata.
4. **Dark is earned, not defaulted** — dark mode is a choice made for the metaphor (deep rock layers feel dark), not because "dev tools are dark."
5. **One decisive idea per fold** — long scroll with deliberate pacing. Each section lands one concept before the next arrives.

## Accessibility & Inclusion

WCAG AA minimum. The geological metaphor is visual — ensure color contrast ratios are met, especially for layered/strata elements. Consider reduced motion for scroll-triggered reveals.
